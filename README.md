# 数据可视化实验平台

一个基于matplotlib和ECharts的数据可视化项目，集成了多个图表类型和交互功能。

## 项目简介

本项目是一个综合性的数据可视化平台，包含：
- **Python数据处理**：使用matplotlib进行数据分析和可视化
- **Web交互界面**：基于ECharts的现代化Web图表展示
- **多主题切换**：支持三种不同的视觉主题
- **响应式设计**：适配桌面端和移动端

## 功能特性

### 🎨 主题配色方案
- **商务专业风**：深蓝色系，专业稳重
- **科技未来风**：蓝绿色系，现代科技感  
- **简约清新风**：绿色系，清爽自然

### 📊 集成图表类型

1. **汇率走势对比图**
   - 2017年7月与2019年7月美元/人民币汇率对比
   - y轴范围：6-7，精确显示汇率数据
   - 点划线/长虚线样式，精确还原实验数据

2. **产品销售对比图**
   - 产品A与产品C各季度销售额对比
   - 区域填充效果，展示销售差异
   - 自动适应数据范围（153-7674）

3. **散点图（颜色映射）**
   - 支持4种专业色系映射（Viridis、CoolWarm、Tab10、Set3）
   - 随机数据生成，展示颜色映射效果

4. **温度变化趋势图**
   - 每日最高温度与最低温度走势
   - 三角形标记，包含最高温度标注

5. **数学函数可视化**
   - 平滑曲线展示y=x²函数
   - ggplot主题风格

### 🔧 交互功能
- **实时主题切换** - 三种主题风格即时切换
- **颜色映射配置** - 四种专业色系动态切换
- **字体大小调节** - 10px-16px范围自由调整
- **响应式设计** - 完美适配桌面、平板、手机

## 详细功能介绍

### 图表技术要点

#### 第一张图表：汇率走势图
- **数据范围**：y轴固定为6-7，便于观察细微汇率变化
- **线条样式**：
  - 2017年：点划线（linestyle='-.'），绿色，线宽1
  - 2019年：长虚线（linestyle='--'），RGB(0.8,0.2,0.4)，线宽3
- **精确还原**：完全按照实验要求实现样式效果

#### 第二张图表：产品销售对比图
- **数据填充**：使用fill_between填充产品A与C之间的差异区域
- **标记样式**：圆形标记，区域填充透明度0.3
- **颜色配置**：浅蓝色(#ADD8E6)填充，增强视觉效果

#### 第三张图表：散点图
- **颜色映射**：支持Sequential类型的Blues色系
- **随机数据**：使用NumPy生成可复现的随机数据
- **交互功能**：可切换多种专业色系映射

### 主题系统

项目实现了完整的主题切换系统：
- **CSS变量**：使用CSS Custom Properties实现动态主题
- **JavaScript控制**：通过JavaScript更新主题变量
- **一致体验**：所有图表和UI元素同步切换主题
